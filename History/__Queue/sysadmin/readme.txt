V1.8 版本更新
--------------------
1. 列表显示改进css，内容太多的时候只显示一行，后面将用"..."省略；
2. 修改网址参数，尽量避免与数据库表字段重名；
3. 修复编辑功能中文本框对数值型内容验证过程不能输入0的问题；
4. 修正“信息列表样式控制”生成的css多一个分号的问题；
5. 修正点击修改密码将会让整个页面跳转的错误；
6. 列表显示将对html标签进行处理，原样输出内容；
7. 修复文件上传控件在必须上传文件限制下，修改信息的时候要求重新上传文件的错误；


V1.7 版本更新
--------------------
1. 修正列表删除的时候提示“数据未配置”的错误；


V1.6 版本更新
--------------------
1. 超级管理员编辑树结构添加删除功能；
2. 字段外联功能，已经外联字段添加已经外联的提示；
3. 休正字段设置为“是否单选”时显示为“性别单选”的错误；
4. 休正列表分页点击其他页后页面报错问题；
5. 编辑页面，如果是操作登录表，将根据设置对密码字段进行相应的加密操作后写入数据库；
6. 后台字段设置中控件类型添加密码框；


V1.5 版本更新
--------------------
1. 配置文件添加 <httpRuntime maxRequestLength="409600" executionTimeout="3600"/> 节点，避免上传大文件（>4M）时直接导致页面无法显示；
2. 后台菜单添加默认是否展开菜单标记；
3. 修改dal中部分可能出现的转换异常；
4. 修复数据库主键使用自动编号从列表中点击编辑功能进入后提示“参数错误”的问题；
